
# Blueprint: Lotto Number Generator

## Overview

This is a simple web application that generates random lottery numbers.

## Features

*   Generate 6 unique random numbers between 1 and 45.
*   Display the generated numbers.
*   Keep a history of generated numbers.

## Current Plan

*   **HTML:** Create the basic structure of the application with a title, a place to display the numbers, a button to generate them, and a history section.
*   **CSS:** Style the application for a clean and modern look, ensuring it is responsive.
*   **JavaScript:** Implement the logic for generating unique random numbers and updating the display and history.
